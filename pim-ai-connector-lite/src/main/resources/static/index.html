🧩 API & Integration für Entwickler & Projektmanager

Der PIM-AI-Connector ist eine REST-basierte Middleware. Die Integration ist einfach und erfolgt über HTTPS mit JSON-Daten.

🔐 Authentifizierung

Alle Anfragen benötigen einen API-Key im Header:
X-API-Key: demo-key (für Demozwecke)
Eigene Keys erhalten Sie nach Projektstart.

📥 Endpunkt: Nur Text (ohne Bild)

POST /api/enrich/gpt/{sprache}

Beispiel:

curl -X POST https://pim-ai-connector-lite.onrender.com/api/enrich/gpt/de \
  -H "Content-Type: application/json" \
  -H "X-API-Key: demo-key" \
  -d '{
    "p1": "Nike",
    "p2": "Laufschuhe",
    "p3": "Schwarz mit roten Akzenten",
    "p4": "Synthetik und Mesh",
    "p5": "EU 42"
  }'

🖼️ Endpunkt: Text + Bildanalyse

POST /api/vision/analyzeAndText/{sprache}

Zusätzlicher Parameter:

imageBase64 – Base64-kodiertes JPG/PNG-Bild

📤 Request-Body (Beispiel)

{
  "p1": "Nike",
  "p2": "Laufschuhe",
  "p3": "Schwarz mit roten Akzenten",
  "p4": "Synthetik und Mesh",
  "p5": "EU 42",
  "imageBase64": "BASE64_STRING"
}

✅ Response (Beispiel)

"Diese schwarzen Nike-Laufschuhe aus atmungsaktivem Mesh und Synthetik bieten optimalen Komfort für sportliche Aktivitäten."
